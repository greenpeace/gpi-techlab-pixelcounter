How do gp links work?

A gp/ short link is a way for somebody to type a short, memorable link such as gp/mail rather than a hard to remember link into their browser.

The user's browser uses the chrome extension to resolve the link to greenpeace.org servers. This server then redirects the browser to the full URL.
